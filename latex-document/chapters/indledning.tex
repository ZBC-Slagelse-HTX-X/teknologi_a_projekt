\subsection{Problemidentifikation}
I det moderne, krise- og krigsprægede samfund, står vi som samfund overfor en række udfordringer der påvirker vores allesammens liv, livskvalitet og fremtidige muligheder. De mest problemfyldte områder identificere vi som: 
\begin{enumerate}
    \item Økonomi: Stigende priser, lavere købekraft og en gennerel usikkerhed vedrørende privatøkonomi.
    \item Miljø og klima: Behovet for bæredygtige løsninger af energi, affaldshåndtering og reduktion af CO2-udslip.
    \item Geopolitiske ustabilitet og krig: De internationale spændinger, krige og konflikter trure global sikkerhed, og herfor påvirker energipriser, inflation og forsyningskæder.
\end{enumerate}

Oventstående er de overordnede problemområder, som vi identificerer præger vores samfund i dag. Disse problemer har allesammen direkte konsekvenser for alle individer i samfundet, og det er derfor vigtigt at vi som samfund tager hånd om disse udfordringer, og arbejder os henimod løsninger, hvor vi kan eksempeltvis finde løsninger der kan optimere borgernes privatøkonomi, sparepenge og derved forbedre livskvaliteten for den enkelte borger.
